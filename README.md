一个基于 Spring Boot 2.1.0 、 Mybatis plus、 JWT、Spring Shiro、Redis、Vue的前后端分离的后台管理系统

#### 项目结构
项目采用按功能分模块的开发方式，结构如下

- `lxadmin-common`  

- `lxadmin-system`  

- `lxadmin-logging`  

- `lxadmin-tools`  

- `lxadmin-generator`  

#### 详细结构

```
- lxadmin-common 公共模块
```
